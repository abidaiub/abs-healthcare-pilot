## Appointment Management: Navigation Map

### 1. Main Menu Structure
*   **Appointments**
    *   `Dashboard`: Today's overview.
    *   `Booking`: New appointment entry.
    *   `Queue Monitor`: Live public display.
    *   `Setup`:
        *   `Doctor Schedules`: Shifts and slot durations.
        *   `Holiday Calendar`: Global and branch holidays.
        *   `Leave Management`: Doctor-specific leave.
    *   `Waitlist`: Manage pending requests.
    *   `Reports`:
        *   `Daily Schedule`: Print for doctors.
        *   `No-Show Analysis`: Performance tracking.
        *   `Waiting Time Report`: Efficiency metrics.
