## Doctor Worklist & Encounter: User Workflow

### 1. Starting the Day (Doctor)
1.  Sign in to ABSHealthcareLite.
2.  Navigate to `Doctor Worklist`.
3.  Filter by `Branch` and `Department` (if multi-assigned).
4.  View the total count of `Waiting` patients.

### 2. Conducting a Consultation (Doctor/Nurse)
1.  (Nurse) Select patient from worklist and enter `Vitals`.
2.  (Doctor) Click `Call` for the next patient.
3.  System opens the `Encounter Dashboard`.
4.  Review `Patient History` and `Previous Prescriptions`.
5.  Record `Clinical Assessment` (Chief Complaint, History).
6.  Enter `Provisional Diagnosis` using ICD-10 search.
7.  Write `SOAP Notes`.
8.  Click `Open Prescription` to add medications.
9.  Click `Open Investigation` to order lab tests.

### 3. Closing the Encounter (Doctor)
1.  Navigate to `Follow-up` tab.
2.  Select `7 Days` and click `Schedule`.
3.  Review the `Encounter Summary`.
4.  Click `Close Encounter`.
5.  Select `Completion Reason: Consultation Finished`.
6.  System moves patient to `Completed` list and updates the `Worklist`.

### 4. Telemedicine Session (Doctor)
1.  Select patient from `Telemedicine Queue`.
2.  Click `Start Session`.
3.  Integrated video window opens.
4.  Record encounter details while maintaining the video call.
5.  Click `End Session` and `Close Encounter`.
